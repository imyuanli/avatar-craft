## Develop

This project is built with `Umi` + `tailwindcss` + `Antd`.

```sh
# 1. Clone project
git clone https://github.com/imyuanli/avatar-craft.git

# 2. Install dependencies
yarn install

# 3. Run
yarn dev
```
